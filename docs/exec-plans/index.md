# Exec Plans

## 概述

本目录保存执行计划、完成归档和技术债务。Agent 可以通过这里判断当前关注点、历史决策和待偿还事项，而不需要依赖口头上下文。

| 文件名 | 摘要 | 最后更新 |

| `completed/index.md` | 已完成执行计划与设计稿归档。 | 2026-03-14 |
| `tech-debt-tracker.md` | 当前已知技术债务、风险和建议优先级。 | 2026-03-14 |
